Tools We Used

Spring Boot

An open-source Java-based framework used to create stand-alone, production-grade Spring-based applications. It simplifies the development process by providing a set of pre-configured settings.

Thymeleaf

A modern server-side Java template engine for both web and standalone environments. It allows for natural templating, meaning templates can be viewed as static prototypes but still be used as working templates.

MySQL

An open-source relational database management system. It's known for its quick processing, proven reliability, ease, and flexibility of use. We used MySQL to store and manage our application's data efficiently.

Java

A versatile, object-oriented programming language known for its portability across platforms. Java is widely used for building enterprise-scale applications and is the foundation for technologies like Spring Boot and Thymeleaf.
